== Description ==

The Mesmerize Companion plugin adds drag and drop page builder functionality to the Mesmerize theme.

The Mesmerize Companion plugin features include:

- Beautiful ready-made homepage
- Drag and drop page customization
- 30+ predefined content sections
- Live content editing 
- and many other features
